Thermal noise with curvature external force functions.

This external Langevin force takes the form:

\[F^{\mu}_{\text{LANGEVIN}}(\dot{x}^{\mu})=\sqrt{\left(2 \gamma k_B T\right)} \hat{\xi}(t)\]


This function depends on one parameter:

  • the temperature \(k_B T\)
  • the damping constant \(\gamma\)
