bool md< dim >::test_index

Test if we need to run the indexing algorithm.

This function determines if re-indexing is necessary, based on the position of each particle at the last time of indexing (xsk[]): If no particle has moved by more than \((ssz-rco)/2\), then there is no pair of particles that were more than network.ssz apart, but now less than network.rco apart, so re-indexing is not necessary. Otherwise, re-indexing might be necessary and thus should be done.
