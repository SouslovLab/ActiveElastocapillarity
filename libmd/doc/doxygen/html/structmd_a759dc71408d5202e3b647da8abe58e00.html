Force add/mod superparticle interaction rule.

This function assigns an interaction to particle numbers p1 and p2 of superparticles of type spt, using element interaction from network.library[]. If the given superparticletype does not exist, a new one is added to network.sptypes[]. It returns the index of the superparticletype (which is spt or the index of the newly created one). It does not perform any checks.
