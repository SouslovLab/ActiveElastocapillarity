Periodicity function to be called if dimension d has hard boundary conditions.

Checks if point x is outside the boundary perpendicular to dimension d and, if so, updates its position to respect a hard wall reflection. The particle position is mirrored across the boundary wall.
This function correctly takes into account skewed boundary conditions, and uses the box matrices simbox.Lshear and simbox.vshear to calculate the reflections if simbox.useLshear is true.
