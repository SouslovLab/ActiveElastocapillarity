Periodicity function to be called if dimension d has periodic boundary conditions.

Checks if particle i has crossed the boundary perpendicular to dimension d and, if so, shifts its coordinate in that dimension by multiples of simbox.L[d] so that it is within the bounds (-simbox.L[d]/2,simbox.L[d]/2).
