Shifts the periodic boxes appropriately for sheared BC.

Update the box matrix using the shear velocities. Increments the box matrix \(L_{ij}\) (stored in simbox.Lshear) by \(h\times v_{ij}\) where \(v_{ij}\) is the shear velocity matrix (stored in simbox.vshear) and \(h\) is the integration timestep (stored in integrator.h).
Also shifts \(L_{ij}\) to ensure that it remains within the bounds \(-L_{ii}/2 \leq L_{ij} \leq L_{ii}/2\).
