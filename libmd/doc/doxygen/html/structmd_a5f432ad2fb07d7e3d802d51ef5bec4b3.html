Add a superparticle bond.

This function creates a new superparticle type and assigns it to the superparticle that p1 and p2 belong to. It then adds an interaction between the two particles, using element interaction from network.library[]. If the two particles are not in the same superparticle, or have an interaction already, it does nothing and returns false.
