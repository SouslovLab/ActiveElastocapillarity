The van Zuiden integrator with protected fixed point itterations (makes sure you don't get stuck in a loop)

This function runs the van Zuiden integrator with fixed point iterations and updates the position and velocity for particle i accordingly. Should the integrator not converge after integrator.generations iterations this function stops with the fixed point iterations.
