template<ui dim>
void mpmd< dim >::integrate
finaloverridevirtual

Integrate particle trajectoriess.

This function is the Monge patch analog to md<dim>::integrate() and calculates the particle trajectories.
After integrating (and updating the particle) it call md<dim>::periodicity(), and calculates the geomatric derivatives for the new particle position.

Reimplemented from md< dim >.
