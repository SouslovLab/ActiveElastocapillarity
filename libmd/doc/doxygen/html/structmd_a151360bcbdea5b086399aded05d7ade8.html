Velocity verlet integrator for position (threaded)

This function integrates particle position i using Velocity Verlet.
See md<dim>::thread_vverlet_dx for the particle velocity part.
Velocity Verlet is a second order symplectic integrator.
It is slower than symplectic Euler (see md<dim>::thread_seuler), yet more accurate.
Velocity Verlet takes the following form:

\begin{align}x^{\mu}_{t+1}=&x^{\mu}_{t}+h\dot{x}^{\mu}_{t}+\tfrac{h^2}{2}F^{\mu}_{t}\\ \dot{x}^{\mu}_{t+1}=&\dot{x}^{\mu}_{t}+\tfrac{h}{2}(F^{\mu}_{t}+F^{\mu}_{t+1})\end{align}

Because of the second force ( \(F^{\mu}_{t+1}\)) calculation this function is split in two.
