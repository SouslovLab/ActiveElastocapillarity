Modify a bond.

This function modifies any previous interaction between particles p1 and p2, and turns it into a bond of previously-defined interaction type specified in the third argument. (See the Interactions documentation for more information on interaction types.)

Does nothing and returns false if one of the following holds:

  • no interaction exists between p1 and p2,
  • the interaction specified by the third argument does not exist or was previously removed.

Otherwise, creates a bond of the specified type between p1 and p2, and returns true.
