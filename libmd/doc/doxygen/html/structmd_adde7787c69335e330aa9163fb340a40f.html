Calculate difference in particle positions in certain dimension i by particle index.

This is an overloaded member function, provided for convenience. It differs from the above function only in what argument(s) it accepts.

This version accepts as inputs the coordinates of two points in arrays x1[dim] and x2[dim] and returns the (periodic) distance between them along spatial dimension d.
