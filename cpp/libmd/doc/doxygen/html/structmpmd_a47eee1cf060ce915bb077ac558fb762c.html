void mpmd< dim >::thread_zuiden ( ui  i)

The van Zuiden integrator for Riemannian manifolds (fails for pseudo-Riemannian manifolds)

This function runs the van Zuiden integrator with fixed point iterations and updates the position and velocity for particle i accordingly. This function unlike mpmd<dim>::thread_zuiden_protect(ui i) waits until the integrator has converged.
