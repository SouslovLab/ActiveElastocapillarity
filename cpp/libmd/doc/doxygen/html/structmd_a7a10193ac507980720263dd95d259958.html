void md< dim >::kdtree
inherited

k-d tree indexing algorithm

This indexing function uses a k-d tree (a multidimensional binary tree).
It makes use of the recursive functions md<dim>::kdtree_build() and md<dim>::kdtree_index().
It does not work if there is both shear and a dimension with periodic boundary conditions.
