template<ui dim>
void md< dim >::index
inherited

Find neighbors.

This function finds all the pairs of particles that are within network.ssz of each other and have an interaction. They are then put in each other's skinlist (network.skins[]).
It calls the algorithm specificied by indexdata.method: md<dim>::bruteforce(), md<dim>::kdtree() or md<dim>::cell().
