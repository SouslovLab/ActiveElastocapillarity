Modify superparticle interaction rule.

This function modifies the interaction between particle numbers p1 and p2 of superparticles of type spt, using a new interaction of the given type and with the given parameters. It returns whether the given superparticletype and interaction exist and the particle numbers already had an interaction. If the two particle numbers do not already have an interaction, it is not added and the interaction is not added to network.library[].
